Amazon Chime SDK Firefox Tests

Scenario 1

Tile Errors

Steps

  1. Open this page on Chrome, and hit Start Meeting
  2. Open this page on Firefox, and hit Start Meeting
  3. On Chrome, turn off and on several times your video stream.

Result:An error will be thrown and video tiles might freeze.

Scenario 2

Duplicated Tiles

Steps

  1. Open this page on Chrome, and allow video to start
  2. Open a second Chrome window and start meeting to add a second tile
  3. Open Firefox and start meeting

Result:A duplicated tile will be added for one of the attendees.
This tile also freezes when other users leave.
